## User Flows

### 1. Report Issue Flow
```
[Open App] → [Tap "Report Issue"] → [Take Photo or Select from Gallery]
    ↓
[Location Auto-Detected] → [Can Adjust Pin on Map]
    ↓
[Select Category] (Pothole, Streetlight, etc.)
    ↓
[Add Title and Description]
    ↓
[Review and Submit]
    ↓
[Success! Issue #12345 Created] → [View Issue Page]
```

### 2. Browse Issues Flow
```
[Open App] → [Map View Shows Pins for Nearby Issues]
    ↓
[Toggle to List View] → [See Issues Sorted by Distance or Date]
    ↓
[Apply Filters] (Category, Status, Date Range)
    ↓
[Tap Issue] → [View Issue Details]
    ↓
[Upvote, Comment, or Create Event]
```

### 3. Organize Event Flow
```
[View Issue Detail] → [Tap "Organize Event"]
    ↓
[Event Template Selected] (Cleanup, Meeting, Demonstration)
    ↓
[Fill Event Details] (Date, Time, Location)
    ↓
[Submit Event]
    ↓
[Event Created] → [Share with Nearby Users]
    ↓
[Users RSVP]
    ↓
[Day of Event] → [Check-in, Post Updates]
    ↓
[Mark Complete] → [Upload Before/After Photos]
```
